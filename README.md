# 🚀 ClusterX - AI-Powered Trading Algorithm

## 📌 Overview
**ClusterX** is a high-performance, AI-driven trading algorithm written in Rust. It intelligently integrates multiple trading strategies and leverages machine learning to analyze signals, providing optimal trading decisions. Whether you're a seasoned trader or an algorithmic trading enthusiast, ClusterX is designed to enhance your trading strategies with cutting-edge AI technology.

## 🌟 Features
✅ **Multi-strategy framework** - Seamlessly integrates various trading strategies.

✅ **AI-driven decision making** - Leverages machine learning to analyze and optimize trades.

✅ **Blazing-fast performance** - Built in Rust for speed, efficiency, and reliability.

✅ **Highly customizable** - Easily add or modify trading strategies to fit your needs.

✅ **Scalable & versatile** - Designed for diverse asset classes and market conditions.

## 🛠 Installation
1. Install Rust (if not already installed):
   ```sh
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
2. Clone this repository:
   ```sh
   git clone https://github.com/enzoblain/ClusterX.git
   cd ClusterX
   ```
3. Build the project:
   ```sh
   cargo build --release
   ```

## 🚀 Usage
1. Configure your trading strategie in the `config/your-strategie.toml` file.
2. Configure the algorithm in the `config/config.toml`file.
2. Run ClusterX:
   ```sh
   cargo run --release
   ```
3. Monitor logs and results to fine-tune your strategies and maximize performance.
