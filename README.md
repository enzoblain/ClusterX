# 🌟 ClusterX

ClusterX is an **algorithmic trading platform** designed to streamline the process of **analyzing market structure, backtesting strategies, and executing real-time trades**. The project is currently in its early stages, focusing on **identifying key market patterns** based on **Smart Money Concepts (SMC) / ICT strategies**.

---

## 🚀 Roadmap

1. **📊 Market Structure Analysis** *(Current Stage)*
   - ✅ Retrieve and process candlestick data
   - ✅ Detect key structures: **Fair Value Gaps (FVG), Order Blocks (OB), ...**
   
2. **📈 Backtesting System** *(Upcoming)*
   - 🛠 Implement a framework to test trading strategies using detected structures
   - 📊 Provide analytics and visualization through a dashboard

3. **🤖 Real-time Trading** *(Future)*
   - ⚡ Deploy trading algorithms using SMC-based strategies
   - 💰 Execute trades automatically with risk management

---

## 🛠 Tech Stack

- **Backend:** 🦀 Rust (Tauri)
- **Frontend:** 🎨 Svelte
- **Database:** 🗄 Redis
- **Market Data Source:** 📡 TwelveData API

---

## 🏁 Getting Started

### 📌 Prerequisites
- 🦀 Rust toolchain
- 🟢 Node.js (for frontend development)
- 📦 Cargo and npm/yarn

### ⚙ Installation
1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/ClusterX.git
   cd ClusterX
   ```
2. **Install dependencies:**
   ```sh
   cargo build
   cd frontend && npm install
   ```
3. **Run the development environment:**
   ```sh
   cargo run
   ```
