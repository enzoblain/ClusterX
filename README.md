## 🚀 ClusterX

**ClusterX** is an advanced algorithmic trading bot developed in **C**. It applies the **Smart Money Concept (SMC)** strategy to analyze market conditions and execute trades based on key liquidity levels and market structure.


## Features
- 🧠 **Smart Money Concept (SMC)**: Uses market structure analysis to make informed trading decisions.
- ⚡ **Real-Time Execution**: Executes trades based on real-time market data.
- 💡 **Customizable Parameters**: Adjust key parameters such as risk management, trading pairs, and strategy settings.
- 🔍 **Trade Decision-Making**: Analyzes signals to determine whether to take a trade, set TP/SL levels, or manage positions based on the current market trend and price action.

## Getting Started

### Prerequisites
Before running **ClusterX**, make sure you have the following:
- A C compiler (e.g., `gcc`).
- Required libraries to run the bot.


### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/enzoblain/ClusterX.git
   ```
2. Navigate into the project folder:
   ```bash
   cd ClusterX
   ```
3. Compile the project:
   ```bash
   gcc -o ClusterX main.c -lm
   ```

---

### Running the Bot
To start the bot and begin trading, run:
```bash
./ClusterX
```

---

### Monitoring
Monitor the bot's performance through the log files.
